Ajocaml

Implementation of the aspect join calculus based on JoCaml

Download .zip Download .tar.gz View on GitHub

Aspect JoCaml is an implementation of the aspect join calculus described in "Chemical Foundations of Distributed Aspects" [1].

Downloading

You can download Aspect JoCaml at https://github.com/tabareau/AJoCaml. It requires OCaml 4.02 and JoCaml 4.01.

Examples

The distribution comes with an example of buffer replication.
